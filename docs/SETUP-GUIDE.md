# Setup Guide

Detailed walkthrough for setting up the AI Assistant Workspace. Covers platform-specific instructions, cloud drive configuration, and optional MCP integrations.

> Content will be expanded after personal use validates the design.

## Quick Setup

1. Clone the repo
2. Run `bash setup/init.sh --data-dir "/path/to/your/cloud-drive/ai-assistant-data"` (or without the flag to be prompted interactively)
3. Fill in your details in the memory templates
4. Start Claude Code and say "start my day"
