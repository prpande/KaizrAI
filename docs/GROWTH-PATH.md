# Growth Path

Week-by-week guide to growing your AI Assistant Workspace.

> Content will be expanded after personal use validates the design.

## Week 1
Daily catch-up and wrap-up. Add todos as they come. Get comfortable.

## Week 2
Start logging accomplishments. Run morning automation. Tune prompts.

## Week 3
Add domain-specific prompts. Build team tracking if managing people.

## Month 2+
Generate weekly summaries. Run health checks. Performance review prep.
