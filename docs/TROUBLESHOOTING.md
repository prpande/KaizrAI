# Troubleshooting

Common issues and fixes for the AI Assistant Workspace.

> Content will be expanded as issues are encountered.

## Common Issues

- **"Missing .env file"** — Run `bash setup/init.sh` to create it
- **"Data directory does not exist"** — Check the path in `.env` and verify the cloud drive folder exists
- **"Database locked"** — Close any SQLite browser that has the .db file open
