# Workspace Preferences

## Output Formatting
- Use emoji status indicators: 🔴 critical/blocked, 🟡 in-progress/warning, 🟢 done/healthy, ⚪ low-priority
- Default to markdown tables for structured data
- Keep summaries scannable — 30-second read max for daily briefings

## Todo Defaults
- Default priority: medium
- Default owner: [your-name]
- Stale threshold: 7 days without updates
- Critical stale threshold: 14 days

## Accomplishment Categories
- project-delivery, technical-leadership, incident-response, mentoring, process-improvement, collaboration

## Automation Preferences
- Morning automation: run all steps, skip none by default
- Weekly review: default lookback is 7 days
- Backup frequency: before any bulk operation
